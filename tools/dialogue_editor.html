Files

open folder & pick the directory containing the dialogue JSON files (typically godot/data/dialogues/).

Files appear in the left pane. Click one to edit. Speaker tags, trigger conditions, and on_dismiss flags appear as read-only context. Click conditions on any state to expand the detail.

Editable: line text, choice text, idle-flavor lines. Schema fields are preserved bit-identical on save.

Save (⌘S / Ctrl-S) writes back to the original file on disk. Folder access uses the File System Access API — Chrome and Edge support it on macOS; Safari and Firefox do not yet (you'll see a folder picker error).
